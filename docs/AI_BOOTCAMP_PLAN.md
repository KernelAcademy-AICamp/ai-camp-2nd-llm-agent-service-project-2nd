### 🔹 Phase 4: 프로젝트 실전 (7-9주차)

#### Week 7: 프로젝트 1 - 학습 도우미 챗봇
- **목표**: LLM API를 활용한 맞춤형 학습 지원 챗봇
- **기술스택**: Python, FastAPI, OpenAI/Claude API, Streamlit
- **주요기능**:
  - 요약 및 Q&A 자동화
  - 학습 자료 추천
  - 진도 관리 및 학습 분석

#### Week 8-9: 프로젝트 2 (선택형)

##### 옵션 A: 바이오 논문 분석 플랫폼
- **목표**: PubMed 논문 자동 분석 및 인사이트 제공
- **기술스택**: Python, FastAPI, BioBERT, Vector DB, RAG
- **주요기능**:
  - 논문 PDF 처리 및 메타데이터 추출
  - 유사 논문 검색 및 추천
  - 트렌드 분석 및 시각화

##### 옵션 B: 법률 문서 자동화 시스템
- **목표**: 계약서/법률 문서 자동 생성 및 검토
- **기술스택**: Python, Django, LLM API, OCR, RAG
- **주요기능**:
  - 문서 템플릿 자동 생성
  - 법률 조항 검색 및 추천
  - 리스크 분석 및 체크리스트

##### 옵션 C: AI 디자인 자동화 도구
- **목표**: 멀티모달 AI를 활용한 콘텐츠 자동 생성
- **기술스택**: Python, FastAPI, DALL-E, Canva API
- **주요기능**:
  - 텍스트 + 이미지 통합 콘텐츠 생성
  - 브랜드 가이드라인 적용
  - SNS 콘텐츠 자동화

## 🛠️ 필수 기술 스택

### Backend
- **언어**: Python 3.11+
- **프레임워크**: FastAPI, Django, Flask
- **API**: RESTful API, GraphQL (선택)

### Frontend
- **간단한 UI**: Streamlit, Gradio
- **웹 프레임워크**: React, Vue.js
- **스타일링**: TailwindCSS, Bootstrap

### AI/ML
- **LLM APIs**: OpenAI, Claude, Gemini, Cohere
- **프레임워크**: LangChain, LlamaIndex
- **모델 허브**: Hugging Face
- **벡터 DB**: Pinecone, Weaviate, ChromaDB

### Database & Infrastructure
- **RDBMS**: PostgreSQL, MySQL
- **NoSQL**: MongoDB, Redis
- **컨테이너**: Docker, Docker Compose
- **버전관리**: Git, GitHub
